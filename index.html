Notes

  • Live pages — open any "live" link and let it run full-screen. There are no controls; it's a screensaver. Thumbnails are single frames, so they can't show the slow motion.
  • Wallpaper Engine — download a .zip and use Wallpaper Engine's Open from file to import it. Each exposes a Scheme Color (background) property.
  • How it works — every piece shares one color engine (a slowly drifting root hue with harmony palettes that cross-fade), so colors evolve smoothly and never flicker between schemes.
  • Two Art Nouveau variants — Organic is the current design (clustered vines, foliage, a meeting hub that fades in as the sunbursts converge); Sparse is a preserved earlier look, kept as its own wallpaper.
  • Heartwood — a surreal forest: a winding path that doubles as two trunks, descending from a tapered, gold-dripping ring crown, over a vined forest that fans toward the edges, in an earthy band of greens, golds and umbers. Simple is the original composition, frozen; the flagship continues from it, adding undulating roots and a canopy that grows off the top of the frame.
Source & build on GitHub. Built with no dependencies; runs entirely offline.
